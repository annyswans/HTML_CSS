quando quiser configurar todos os elementos de uma tag mesmo que tenha outras dentros não se utiliza o > (filho). Exemplo: dentro de uma tag main tenho uma tag article, se eu utilizar a tag em CSS main>p será configurado somente o estilo fora de article. Nesse caso não utilizamos o >, e ficaria main p {configuração desejada}, sendo assim configurado todos os p dentro da tag main e article

                             RESPONSIVIDADE 

Utilize a extensão window resizer para verificar o tamanho de tela que começa a afetar o site.
utiliza-se a tag width para criar o tamanho máximo do site, porém é melhor utilizar as tags "min-width e max-width" em seguida adicione a tag margin: auto; para centralizar:

exemplo: 
min-width: 320px;
max-width: 800px;
margin: auto;


para tornar a imagem responsiva basta utilizar a tag nas configurações de estilo "img{width: 100%;" + adicionar a responsividade de picture no HTML.
