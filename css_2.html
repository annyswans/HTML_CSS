Seletores Personalizados

Usando ID com CSS

  • Os seletores são utilizados quando você quer dar uma formatação diferente para uma mesma tag ou tags semelhantes como é o caso da tag title e h1.
  • Para adicionar uma formatação diferente basta ir na tag escolhida e acrescentar a sua frente a tag id e nomeá-la, exemplo: id= "principal". Em seguida vá nas configurações de CSS externo e adicione a tag desejada + #(nome escolhido), exemplo: h1#principal {configurações desejadas} ou #principal {configurações desejadas}.
  • Em HTML "ID" = em CSS é "#"
  • A regra do W3C é que não se pode utilizar 2 seletores ID iguais no HTML.

Usando CLASS

  • Quando é necessário utilizar 2 seletores para uma tag utilizasse o seletor class
  • Para adicionar uma formatação diferente basta ir na tag escolhida e acrescentar a sua frente a tag class e nomeá-la. Em seguida vá nas configurações de CSS externo e adicione a tag desejada + .(nome escolhido) ou somente .(nome escolhido), exemplo: h1.(nome escolhido) {configurações desejadas}.
  • Em HTML "CLASS" = em CSS é "."
  • É possível adicionar mais de uma classe a uma tag, basta adicionar a sua frente sem vírgula, exemplo: <h1 class = "principal destaque"> (principal seria um class configurado com um estilo e destaque outro)

Pseudo classes

  • As pseudo classes estão relacionadas ao seu estado que é representado por ":", exemplo; hover é utilizado para que o estilo mude quando o mouse passa por cima da tag aplicada

  • Outros exemplos de pseudo classes são: visited (visitados), active (click do mouse), checked ( estilizar elementos marcados ou selecionados), empty (elementos vazios) e focus (estilizar elemento em foco, quando selecionado ou ao clicar, para que o usuário saiba onde está selecionando)

Pseudo elementos

  • Pseudo elementos estão relacionados ao conteúdo, e pode ser usado nas id ou class. Os pseudo elementos são representados por "::" e os principais utilizados nas CSS são: before, after, first-letter ou first-line. Exemplo de aviso ao usuário que é um link:
    •                             a::after {
                                content: '[link]',
                            }
                        que ficará:

Modelo de caixas

Modelo box-level

  • Um elemento box-level sempre vai ser iniciado em uma nova linha e irá ocupar a largura total da tela do dispositivo. O elemento box-level mais conhecido é o div e suas variações (main, section, aside). Segue uma lista de exemplos de elementos box-level:
    • h1-h6, header, hr, li, ol, p, pre, section, table, tfoot, ul, video, main, nav, noscript,div, dt, figure, footer, form, dl, fieldset, figcaption,address, article, aside, blockquote, canvas, dd...

Modelo inline-level

  • Um elemento inline-level não inicia uma nova linha, continua do exato ponto onde parou e sua largura vai ocupar apenas seu tamanho relativo ao seu conteúdo. Segue uma lista de exemplos de elementos inline-level:
    • a, abbr, acronym, b, bdo, br, button, cite, code, dfn, em, i, img, input, kbd, label, map, object, output, q, samp, script, select, small, span, strong, sub, textarea, tt, var...

Utilizando shorthand

  • As shorthands servem para simplificar o estilo do corpo, cada tag tem sua ordem de estilo. Segue a ordem de cada tag e seus respectivos shorthands:
  • borden: width, style e color. Exemplo: "borden: 10 px solid orange;"
  • padding: top, rigth, bottom e left. Exemplo: "padding: 10 px 10px 10px 10px;", quando se tem dados iguais é possível simplificar apenas com "padding: 10px;". Quando são duas medidas ficaria "padding: 10 px 20px;"
  • margin: top, rigth, bottom e left. Exemplo: "margin: 10 px 10px 10px 10px;" ou quando desejar centralizar basta utilizar "margin: auto 10px auto 10px;"
  • outline: width, style e color. Exemplo: "outilne: 5px auto blacl;"

Tags de agrupamento

  • Servem para organizar de forma semântica o site, existem 3 tags sendo elas: header (cabeçalho do site), main (corpo do site) e footer (rodapé).

Sombras em caixas

  • O sombreamento de caixas é representado pela shorthand box-shadow, as ordens das informações são deslocamento horizontal, deslocamento vertical, espalhamento e sombra. Segue exemplo: "box-shadow: 1px 1px 1px black;"
  • obs: para ficar mais fácil a visualização do box-shadow é recomendado utilizar o expecionar do navegado para visualizar e alterar o visual do site.
  • Diferença entre div e span, div pula uma linha e span cria uma caixa, porém segue na mesma linha
  • Para esconder uma tag p dentro da div basta adicionar o <, que diz que a tag p é filha de div, exemplo: div < p.
  • display: none (não aparecer as informações dentro daquela tag). Exemplo: div > p {display: none;}, ou seja, qualquer tag p adicionada dento de uma div não irá monstrar o conteúdo. Caso queira que o texto apareça quando passar o mouse em cima basta adicionar div > p {display: block;}
  • display: block
  • text-decoration: none; tira o sublinhado do texto ou link
