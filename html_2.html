LINKS EXTERNOS

Indicações do link

basta indicar o nível do link com a tag rel

  • next: usado para indicar que o link leva a página seguinte da página atual
  • prev: usado para indicar que o link leva a página anterior a página atual
  • author: indica que o link leva para o site do autor do artigo atual
  • external: indica que o link leva a um site diferente do atual
  • nofollow: indica que é um link externo não endossado, tipo link pago
  • self: para abrir na própria aba

RESPONSIVIDADE DE IMAGEM

  • Utilize a extensão 'window resizer' para verificar o tamanho de tela que começa a afetar o site.
  • utiliza-se a tag width para criar o tamanho máximo do site, porém é melhor utilizar as tags "min-width e max-width" em seguida adicione a tag margin: auto; para centralizar:
    •                         exemplo:
                                min-width: 320px;
                                max-width: 800px;
                                margin: auto;
  • para tornar a imagem responsiva basta utilizar a tag nas configurações de estilo "img{width: 100%;}" + adicionar a responsividade de picture no HTML.
