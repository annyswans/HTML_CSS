Segue abaixo o significado de cada tag dentro da tabela:

  • table: tag utilizada para criar uma tabela, normalmente se adiciona border em seguida para criar as bordas da tabela
  • thead: é o cabeçalho da tabela
  • tbody: significa "corpo da tabela" é usado em conjunto com a tag thead para que a tabela fique mais harmônica
  • tfoot: rodapé da tabela
  • tr: são as linhas da tabela
  • th: é o cabeçalho da coluna
  • td: são os dados da tabela
