IFRAME

  • Representa um contexto de navegação aninhado, efetivamente incorporando outra página HTML para a página atual.

  • Resumo HTML 2

  • Nem todos os dispositivos são compatíveis com o iframe, então o recomendado é colocar um link dentro do iframe
  • É possível adicionar parametros de rolagem para o iframe usando a tag scolling= "", sendo usado as configurações de "auto" (rolagem quando o site excede), "yes" (aparece tela de rolagem mesmo sem exceder) e "no" (não permite a rolagem)
  • A configuração de frameborder dentro do iframe só aceita 2 númerações, sendo ela 0 (sem borda) e 1 (com borda). Para o tamanho é necessário configurar pela tag border.
  • Para adicionar várias páginas pessoais e abrir no iframe basta adicionar a subclasse name em iframe, após basta adicionar target="nome escolhido"
